Responsive Typography
Mobile h1 = text-3xl md:text-4xl

Mobile h2 = text-2xl md:text-3xl

Mobile body = text-base (already default)

Desktop h1 = md:text-4xl or lg:text-5xl if desired (override above)

Main content container = container mx-auto px-4 sm:px-6 lg:px-8

container applies responsive max-widths (default: 640px, 768px, 1024px, 1280px, 1536px)

mx-auto centers it horizontally

px-4 gives horizontal padding on mobile, increases on tablet/desktop

Alternative: Custom Max-Width (more control)
Wide container = max-w-7xl mx-auto px-4 sm:px-6 lg:px-8 (common for marketing sites)

Narrow container (reading content) = max-w-3xl mx-auto px-4 (for blog posts, docs)

Full-bleed sections = w-full with inner container for constrained content
